# Story Export — KDP-Compliant Publication

Export for Amazon KDP: **$ARGUMENTS**

> **PREREQUISITES:**
> - Story at v2+ (post-improvement)
> - Layout generated by `/story-layout`
> - Illustrations in `illustrations/[slug]/`

## Constants

- OUTPUT_DIR = "output/"
- KDP_TRIM = "8.5x8.5"
- EBOOK_PRICE = "$2.99"
- PAPERBACK_PRICE = "$12.99"
- BOOK_1_EBOOK_PRICE = "$0.99" (series loss leader)
- ROYALTY_RATE_EBOOK = 0.70 (70% for $2.99-$9.99 range)
- ROYALTY_RATE_PAPERBACK = 0.60 (60% minus printing cost)

## Workflow

### Phase 1: Compile Exportable Package

For each approved story:

1. Read story markdown (v2 or v3) with frontmatter
2. Read layout files from `output/[slug]/`
3. Read illustration images from `illustrations/[slug]/spreads/`
4. Read character bible from `characters/[slug]/`
5. Read series metadata from `series/[slug]/` (if applicable)

```markdown

### Content
- [ ] Copyright page includes AI disclosure
- [ ] All text within safe zone (0.25" from trim)
- [ ] No text in gutter area
- [ ] Cover text is readable at thumbnail size

### Metadata

- [ ] 7 keywords entered (all relevant, no keyword stuffing)
- [ ] 2 BISAC categories selected

- [ ] Series name and position set (if applicable)
```

## Key Rules

- **AI disclosure is MANDATORY** — Amazon requires it since 2023
- **Fixed-layout EPUB only** — reflowable is wrong for picture books
- **Print-ready means 300dpi** — check every file before packaging
- **Bleed is required** — 0.125" on all sides
- **Spine math**: width = 0.002252" × page_count (for white paper)
- **Book 1 at $0.99** — this is the industry standard funnel strategy
- **Large file handling**: If Write fails, use Bash fallback
